# 任务状态查询系统

一个基于PHP和JavaScript的任务状态查询与管理系统，支持多任务文件监控、实时通知和任务领取功能。

## 功能特性

### 主查询页面 (index.php)
- 显示多个任务文件的状态卡片
- 按更新时间排序，优先展示最新修改的任务
- 实时检测任务文件变化，自动播放通知声音
- 支持两种通知模式：循环通知 / 单次通知
- 可配置显示的任务数量（1-20条）
- 点击任务卡片可跳转至对应的任务管理页面

### 任务领取系统 (子目录)
- 任务创建与管理：支持批量创建任务
- 进度跟踪：每个任务可包含多个进度节点
- 任务领取：用户可领取任务并记录领取时间
- 完成状态：标记任务完成并记录完成时间
- 分页浏览：支持分页查看大量任务
- 隐藏任务管理：支持任务归档（超过100条自动隐藏）
- 数据导出：支持导出为Excel格式
- 自动保存：普通操作自动保存，重要操作需密码验证

### 配置管理
- 通知模式配置（循环/单次）
- 显示任务数量配置
- 配置文件持久化存储

## 安装与运行

### 环境要求
- PHP 7.0+
- Web服务器（Apache/Nginx）
- 支持JavaScript的现代浏览器

### 快速开始

1. 将项目文件部署到Web服务器根目录

2. 确保以下目录和文件有写入权限：
   - `config.ini` - 配置文件
   - `01/tasks.json`, `1/tasks.json` 等任务数据文件

3. 访问主页面：
   ```
   http://your-domain/index.php
   ```

4. 访问任务管理页面：
   ```
   http://your-domain/01/index.html
   ```

## 使用说明

### 主查询页面操作
1. **查看任务状态**：页面会自动显示所有任务文件的更新时间
2. **刷新数据**：点击"刷新数据"按钮手动刷新
3. **切换通知模式**：点击"循环通知"/"单次通知"按钮切换
4. **调整显示数量**：修改数字输入框，按回车或失焦后生效

### 任务领取系统操作
1. **功能菜单**：点击展开设置面板
2. **新增任务**：设置任务数量和进度数量后点击"新增"
3. **重置任务**：点击"重置"清空所有任务并重新创建
4. **领取任务**：在进度项中输入姓名即可领取
5. **完成任务**：勾选进度项右侧的开关标记完成
6. **查看隐藏任务**：点击"查看隐藏"显示超过100条的任务
7. **导出数据**：点击"导出"下载Excel文件
8. **保存修改**：点击"保存修改"保存所有变更（重要修改需密码：3）

## 配置说明

### config.ini 配置项

```ini
notify_mode = loop    # 通知模式：loop（循环）或 once（单次）
task_limit = 6        # 显示任务数量限制（1-20）
```

### config.php 可配置项

```php
define('DEFAULT_TASK_LIMIT', 4);    // 默认显示任务数量
define('DEFAULT_NOTIFY_MODE', 'loop'); // 默认通知模式

$filesToCheck = [
    '01/tasks.json',
    '1/tasks.json',
    // 添加更多任务文件...
];
```

## 技术实现

### 前端技术
- HTML5 / CSS3
- TailwindCSS 3.x
- Font Awesome 图标库
- 原生JavaScript（ES6+）

### 后端技术
- PHP 7.0+
- JSON数据存储
- 文件系统操作

### 核心特性
- **实时监控**：每3秒检测一次文件变化
- **乐观锁**：防止并发写入冲突
- **自动保存**：输入操作500ms后自动保存
- **版本冲突处理**：检测到冲突自动重新加载

## 注意事项

1. 重要数据修改需要密码验证（默认密码：3）
2. 任务数量超过100条会自动归档到隐藏区域
3. 通知音效需要用户交互后才能自动播放（浏览器安全限制）
4. 建议定期备份 tasks.json 文件
